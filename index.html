La ligne de commande

Gestion de documents

Pour afficher le manuel de commande ls : man ls
Pour créer un dossier je dois utiliser cette commande : mkdir nomdudossier
Pour créer un fichier, je dois utiliser cette commande : touch nom_du_fichier
Pour voir ce que contient un dossier (fichier, dossier, etc) : ls
Pour tout voir, même les fichiers cachés : ls -la
Pour voir les fichiers version longue : ls -a
Idem avec les fichiers cachés : ls -al
Pour afficher le chemin du dossier courant : pwd
Pour naviguer dans les dossier : cd nom_des_dossiers
Pour revenir dans le dossier précédent : cd ..
Pour déplacer un dissier ou un fichier : mv nom-du fichier chemin
Pour démarrer un logiciel : nom_du_logiciel nom_du_fichier
Pour démarrer un logiciel en super-utilisateur : sudo -i nom_du_logiciel nom_du_fichier

Mise à jour et installation

Pour passer en mode super utilisateur : sudo -i + mdp utilisateur
Pour voir les mises-à-jour disponibles pour les paquets : apt update
Pour les mettre à jour : apt upgrade
Pour rechercher les paquets contenant un terme : apt search terme
Pour installer un paquet : apt install nom_du_paquet
Pour installer un logiciel : dpkg -i nom_du_logiciel

Utiliser Nano

Pour éditer un texte avec Nano : nano nom-du-fichier
Pour copier un texte : Shift + inser
Pour remplacer des caractères, un mots, une phrase, etc : alt + R
